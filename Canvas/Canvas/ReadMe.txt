Shapes:
All shapes draw as described in the prompt.  Drawing a Polygon will prompt for the number of sides of the shape to draw.  
Shapes that will be outside the bounds will not be drawn.

Single Select:
clicking on a point that shares more than one object will iterate the selection through all connected objects.
Holding shift and selecting the same object over and over will select the next closest unselected shape until none remain.

Move:
Nothing will happen if you try to move something out of bounds.

Nudge:
The arrow keys are used to nudge selected objects.

Save Data:
The data string to save a drawing will appear in the save data text box at the bottom. 

Save Selected:
Will only save selected(red) objects and their positions.

Load/insert Data:
Enter a saved data string into the load data prompt to load/insert a drawing.

This tool is only compatible with Firefox and Chrome.
